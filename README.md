# 📰 Fake News Detection using Machine Learning

An end-to-end Machine Learning project that classifies news articles as **Real** or **Fake** using Natural Language Processing (NLP) and Machine Learning techniques.

---

## 📌 Project Overview

Fake news spreads misinformation rapidly through social media and online platforms. This project aims to automatically classify news articles into **Real** or **Fake** categories using TF-IDF feature extraction and Machine Learning algorithms.

---

## 🎯 Problem Statement

Develop a Machine Learning model capable of accurately detecting fake news articles from textual content, helping users identify misleading information and supporting reliable information consumption.

---
